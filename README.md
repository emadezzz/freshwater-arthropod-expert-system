# Freshwater Arthropod Expert System

Rule based expert system for freshwater arthropod identification developed using CLIPS and symbolic Artificial Intelligence techniques.

## Overview

This project was developed as part of an Intelligent Systems module and focuses on identifying freshwater arthropods through a rule based expert system.

The system guides users through a sequence of questions and uses symbolic AI techniques to determine the most likely organism. The project applies decision trees, forward chaining, and knowledge based reasoning to simulate expert decision making.

---

## Features

- Rule based expert system
- Built using CLIPS
- Forward chaining inference
- Knowledge representation using rules
- Decision tree based identification
- Input validation
- Interactive question and answer system
- Freshwater arthropod classification
- Explainable AI approach
- Demonstration video included

---

## Technologies Used

- CLIPS
- Rule Based Artificial Intelligence
- Symbolic AI
- Expert Systems
- Knowledge Representation
- Decision Trees

---

## System Process

The system asks users a series of structured questions related to:

- Body shape
- Number of legs
- Habitat characteristics
- Physical features
- Classification properties

Based on user responses, inference rules are triggered until an identification result is produced.
